This board is a board I made wanting to test out some other woods that I haven't used before. I also tested out making a double-sided board. It surprisingly came out looking good. This side uses the same woods as the other except for the center and two of the stripes towards the outside. So, there is walnut woods, bloodwood, and tiger wood. On this side we also created a juice groove.

This side is the same as the other side except that it uses wenge wood instead of walnut and it doesn't have a juice groove.

This is a very special handle that I made for this two-sided board. Unlike the normal 45-degree angle that we use we did two 45-degree angles having them meet int he middle to make it so we'll be able to pick it up from either side easily.
